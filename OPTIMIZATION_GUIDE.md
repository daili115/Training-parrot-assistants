# 鹦鹉学舌助手 - 老年用户优化指南

## 🎯 优化目标

针对不会打字的老年人群体，优化鹦鹉训练PWA应用，使其更加友好、易用。

## 📋 主要优化内容

### 1. 简易模式 (Easy Mode)

#### 新增功能
- **一键切换**：点击"简易模式"按钮即可切换界面
- **大字体支持**：提供正常、大、超大三种字体大小
- **语音辅助**：操作时自动语音播报提示
- **简化界面**：隐藏复杂功能，只保留核心操作

#### 简易模式特点
- ✅ 超大按钮设计（最小44×44px，符合触摸标准）
- ✅ 底部导航栏，操作更直观
- ✅ 减少模态框层级，简化流程
- ✅ 大字体显示，提升可读性
- ✅ 语音提示，辅助操作理解

## 📱 使用指南

### 开启简易模式
1. 点击右上角的"简易模式"按钮
2. 在悬浮菜单中调整设置：
   - 字体大小：正常/大/超大
   - 语音辅助：开启/关闭
   - 简化界面：开启/关闭
3. 界面会自动切换为简易模式

### 简易模式操作
1. **录音**：点击底部"录音"按钮 → 选择标签 → 保存
2. **训练**：点击底部"训练"按钮 → 自动播放
3. **查看技巧**：点击底部"技巧"按钮 → 选择分类
4. **查看勋章**：点击底部"勋章"按钮

### 语音辅助
- 开启后，操作时会自动语音播报
- 适合视力不佳的用户
- 可在简易模式设置中关闭

### 字体大小
- 根据视力选择合适的字体大小
- 超大字体适合视力较弱的用户

## 🔧 技术实现

### 状态管理
```typescript
interface EasyModeContextType {
  isEasyMode: boolean;           // 简易模式开关
  fontSize: 'normal' | 'large' | 'extra-large';  // 字体大小
  voiceAssist: boolean;          // 语音辅助
  simplifiedUI: boolean;         // 简化界面
}
```

### 持久化存储
- `parrot_easy_mode`：简易模式状态
- `parrot_font_size`：字体大小
- `parrot_voice_assist`：语音辅助状态
- `parrot_simplified_ui`：简化界面状态

### 语音合成
- 使用浏览器原生 `speechSynthesis` API
- 自动选择中文语音
- 支持语速和音量调节
